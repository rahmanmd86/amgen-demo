# Environment setup

- Java 1.8
- Maven 3.6.0
- Selenium 4.16.1
- TestNG 7.8.0
- Chrome 120.0.6099.130 (Windows 11 Pro)

# Executing tests

- Clone the project
- Install dependencies and run tests `mvn clean install`
- Only install depedencies and skip tests `mvn clean install -DskipTests`
- Only Run tests `mvn test`
